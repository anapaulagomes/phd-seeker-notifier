Notify about PhD positions found using [PhD Seeker](https://github.com/Aghababaei/PhD-Seeker).

## Development

#### Create a virtual environment

```bash
python -m venv venv
source venv/bin/activate
```

#### Install the dependencies

```bash
pip install -r requirements.txt
```
